Khakehashi
===

Hi! I am a dynamic WordPress consultancy theme built on the _s starter foundation. I've been "hacked" and transformed into a high-performance tool for educational and professional consultancies.

Instead of using heavy page builders, I rely on a "pure code" approach using Tailwind CSS and Advanced Custom Fields (ACF) for a lightweight, professional experience.

=== Features Included:
Custom Consultancy Logic: Specialized templates for services and team profiles.

Tailwind CSS Integration: Modern, utility-first styling for rapid UI development.

Lean Templates: Well-commented, HTML5 templates based on Underscores.

Custom Header & Tags: Found in inc/custom-header.php and inc/template-tags.php to keep code clean.

Mobile Navigation: Responsive dropdown menu handled by js/navigation.js.

=== Installation
Requirements
Node.js (for Tailwind and asset compilation)
Composer (for PHP linting and dependency management)
